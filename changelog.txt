## v1.3.0 — 03.09.2025

- 🐞 Виправлено помилку з командою /tomorrow (порожній розклад).

## v1.8.6 — 07.09.2025
- 📲 Додано inline-режим: отримання розкладу у будь-якому чаті через `@pnpu_schedule2025_bot`.
  - Якщо групу обрано — доступні швидкі картки: «на сьогодні», «на завтра», «на 7 днів», «на наступний тиждень».
  - Якщо групу не обрано — показується підказка з переходом у бот.
- У команді /calendar тепер додатково показується посилання для iOS (раніше був лише файл для Android).
- ⏳ При використанні команди /calendar або кнопки «Завантажити календар» тепер показується повідомлення «Генеруємо календар...», яке згодом змінюється на готовий результат.
- 📂 Для Android календар надсилається як файл.
- 🍏 Для iOS одразу показується посилання для підписки.
- 🛠️ Кнопка «Завантажити календар» перенесена в головне меню (раніше була у налаштуваннях розсилки).

## v1.8.8 — 08.09.2025
